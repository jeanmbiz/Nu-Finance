# Sobre o projeto

Aplicativo de controle financeiro em React com as funcionalidades:

- Adiciona e remove transação
- Cálculo do saldo podendo ser positivo ou negativo
- Filtro das movimentações
- LandingPage

# Tecnologias utilizadas
## Front end
- React
- useState
- StyledComponents
- React Icons
